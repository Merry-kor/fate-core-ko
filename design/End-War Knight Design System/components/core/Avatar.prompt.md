**Avatar** — a character portrait in a steel imperial frame; use `ring="active"` for whoever currently holds the stage.

```jsx
<Avatar src={anya.img} name="안야 슈탈" size={64} ring="active" />
<Avatar name="이반" shape="frame" ring="crimson" badge={3} />
```

`shape`: circle | frame. `ring`: none | gold | crimson | active.
