**Badge** — pill marker for divisions, aspect-type tags (`#단기`/`#장기`), and counts.

```jsx
<Badge tone="crimson" variant="solid">제1사단 스피어헤드</Badge>
<Badge tone="violet" size="sm">#장기</Badge>
<Badge tone="gold" variant="outline">발현 ×2</Badge>
```

Tones: gold, crimson, steel, success, tie, fail, violet, teal, blue. Variants: solid, soft, outline.
